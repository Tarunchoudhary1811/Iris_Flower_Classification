# Iris_Flower_Classification
A Machine Learning project that predicts the species of an iris flower (Setosa, Versicolor, or Virginica) using sepal and petal measurements from the Iris dataset.

🛠️ Tech Stack
Python
Pandas
NumPy
Matplotlib
Scikit-learn
Jupyter Notebook
✨ Features
Data preprocessing and visualization
Model training and evaluation
Species prediction
Performance analysis using accuracy and classification metrics
📊 Outcome

Built a classification model that accurately predicts iris flower species, demonstrating the complete machine learning workflow from data preprocessing to model evaluation.
